## Sobre o projeto

Este é o meu portfólio pessoal, criado para apresentar meus estudos e projetos em desenvolvimento backend.  
Usei um visual inspirado em Pokédex para deixar claro que estou em constante evolução como dev.

---

## Tecnologias

- Next.js  
- TypeScript  
- Tailwind CSS  

---

## Como rodar o projeto

```bash
git clone https://github.com/vicbaltazar/pokefolio.git
cd pokefolio
npm install
npm run dev

Depois, acesse: http://localhost:3000

```md
